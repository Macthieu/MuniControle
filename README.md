# MuniControle

MuniControle est l'outil spécialisé de contrôle qualité documentaire de la suite municipale.

## Mission

Vérifier la conformité documentaire (qualité, règles, complétude, traçabilité) avant validation finale.

## Positionnement

- Rôle: outil spécialisé de contrôle.
- Orchiviste reste le cockpit/hub; MuniControle reste un outil autonome dédié.

## Versionnage

- Version initiale de normalisation: `0.1.0`.
- Le projet suit [Semantic Versioning](https://semver.org/lang/fr/).
